# 第 1 章：课程概述


在本课程中，我们将通过以下主要步骤在 Bitcoin 网络上构建基于 zkSNARK 的战舰游戏：


1. 使用Zokrates工具包创建专用的零知识证明电路；
2. 创建 sCrypt 智能合约以在链上运行游戏；
3. 添加友好的用户界面。
